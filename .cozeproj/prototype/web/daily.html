大模型动态

前沿发布

OpenAI 正式发布 GPT-5

OpenAI 官方博客 · 2025-06-15 09:30

OpenAI 在今日凌晨正式发布 GPT-5，新版本在多模态理解、推理能力和代码生成方面均有显著提升。据官方介绍，GPT-5 在多项基准测试中超越了前代产品，并新增了对视频内容的深度理解能力。

开源

DeepSeek 开源 V3 模型

DeepSeek 技术团队 · 2025-06-15 11:00

DeepSeek 宣布开源其最新研发的 V3 大语言模型，该模型在保持高性能的同时大幅降低了推理成本。模型权重和训练代码已托管在 GitHub 和 Hugging Face 平台，供研究者和开发者免费使用。

产品更新

Anthropic 推出 Claude 4 重大更新

Anthropic 新闻中心 · 2025-06-15 14:20

Claude 4 带来多项功能增强，包括更长的上下文窗口（最高支持 50 万 token）、改进的数学推理能力以及全新的「深度研究」模式，可帮助用户完成复杂的多步骤分析任务。

AI 应用

AI 应用层持续升温，开发工具与搜索产品获得资本市场高度关注。

融资

Cursor 完成 3 亿美元 C 轮融资

TechCrunch · 2025-06-15 10:15

AI 代码编辑器 Cursor 宣布完成 3 亿美元 C 轮融资，投后估值达到 30 亿美元。本轮融资由 a16z 领投，资金将用于扩展团队规模和提升模型推理能力。

产品动态

Perplexity 推出企业级搜索解决方案

The Verge · 2025-06-15 16:45

Perplexity 发布面向企业客户的搜索平台，支持私有数据源接入、团队协作和权限管理。该产品旨在与微软 Copilot 和谷歌 Enterprise Search 展开竞争。

行业动态

全球 AI 监管与产业合作持续深化，政策环境逐步明朗。

政策法规

欧盟 AI 法案正式生效执行

Reuters · 2025-06-15 08:00

欧盟《人工智能法案》今日起正式全面生效，成为全球首部综合性 AI 监管法律。该法案对高风险 AI 系统实施严格合规要求，违规企业最高面临全球年营业额 7% 的罚款。
